# 🌐 Dhanush | Futuristic Portfolio

A modern, animated, and fully responsive personal portfolio website built with pure **HTML, CSS, and JavaScript**, featuring glassmorphism UI, interactive animations, and a dynamic 3D background using VANTA.js.

## ✨ Features

🔥 **Futuristic UI Design**
- Glassmorphism cards
- Neon glow effects
- Smooth gradients & blobs

🎯 **Interactive Experience**
- Custom animated cursor glow
- Scroll reveal animations
- Click-to-view project modal

🌌 **3D Animated Background**
- Powered by VANTA.NET (Three.js)
- Interactive network-style motion effect

📱 **Fully Responsive**
- Mobile-first design
- Optimized layout for all screen sizes

🧠 **Clean Project Structure**
- Modular sections: About, Skills, Projects, Resume, Contact

📂 **Downloadable Resume**
- One-click resume download feature
